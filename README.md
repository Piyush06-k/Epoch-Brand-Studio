# Epoch-Brand-Studio

A modern, responsive brand studio website built with **React**, **TypeScript**, and **Vite**. The project is designed to showcase branding services with a clean UI, smooth performance, and responsive design.

## Features

* Modern and responsive user interface
* Fast performance powered by Vite
* Component-based architecture
* TypeScript support
* Optimized production build

## Tech Stack

* React
* TypeScript
* Vite
* CSS
